# ✅ Todo App - Frontend

Aplicación web de gestión de tareas construida con React y Vite.

## 🚀 Tecnologías

- **React** — Librería de interfaces de usuario
- **Vite** — Herramienta de desarrollo ultrarrápida
- **Framer Motion** — Animaciones fluidas
- **CSS** — Estilos personalizados con diseño responsive

## ⚙️ Instalación

1. Clona el repositorio

git clone https://github.com/tu-usuario/todo-frontend.git
cd todo-frontend

2. Instala las dependencias

npm install

3. Arranca el servidor de desarrollo

npm run dev

4. Abre en tu navegador

http://localhost:5173

⚠️ Asegúrate de tener el backend corriendo en http://localhost:8000

## ✨ Funcionalidades

- 🔐 Registro e inicio de sesión de usuarios
- ✅ Crear, completar y eliminar tareas
- 🔍 Filtrar tareas por estado (todas, pendientes, completadas)
- 📊 Contador de tareas en tiempo real
- 🎨 Animaciones fluidas con Framer Motion
- 📱 Diseño responsive para móvil y escritorio
- 💾 Sesión persistente con localStorage
